概述

环境

  • Python 3

方法

xlrd提供了sheet.merged_cells用于获取页面中所有的合并单元格坐标。类似：

坐标从0开始算起，代表第1行到第4行（不包含4），第0列到第一列（不包含1）为合并单元格。
